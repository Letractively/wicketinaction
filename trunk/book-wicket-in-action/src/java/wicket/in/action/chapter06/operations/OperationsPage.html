6.4 Basic operations on components

6.4.1 Changing the visibility

This section shows several examples modifying the visibility of components.

ExampleResult
label.setVisible(false);
label.setVisible(!label.isVisible());
new Label("label", "Now I'm visible!") {
      @Override
      public boolean isVisible() {
        int day = Calendar.getInstance().get(Calendar.DAY_OF_WEEK);
        return day != Calendar.SATURDAY && day != Calendar.SUNDAY;
      }
    }
