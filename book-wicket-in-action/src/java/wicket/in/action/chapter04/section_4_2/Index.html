4.2 A taste of the standard models

These examples show how Wicket's standard models work.

4.2.1 Using the simple Model

The following examples demonstrate the simple Model class.

Static models

add(new Label("firstname", customer.getFirstName()));
add(new Label("lastname", customer.getLastName()));
add(new Label("street", customer.getAddress().getStreet()));
add(new Label("firstname", new Model(customer.getFirstName())));
add(new Label("lastname1", new Model(customer.getLastName())));
add(new Label("street1", new Model(customer.getAddress().getStreet())));
